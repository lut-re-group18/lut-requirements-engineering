## Test MD to PDF conversion (localy)

You need Docker installed on your machine. Then run the following command:

```bash
docker run --rm \
       --volume "$(pwd):/data" \
       --user $(id -u):$(id -g) \
       pandoc/latex:2.19 --output=result.pdf \
       "/data/docs/requirements-management-plan.md"
```
